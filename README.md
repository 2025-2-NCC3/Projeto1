Descrição
<p align="center"> <img src="https://pix4free.org/assets/library/2021-01-20/originals/original_cantina.jpg" alt="Comendaria da Tia" border="0"> </p>

O projeto Comendaria da Tia é um aplicativo mobile destinado a otimizar o processo de compra na cantina da FECAP, eliminando as filas no caixa e proporcionando uma experiência ágil e prática para os alunos. Através do app, os estudantes podem realizar pedidos e pagamentos antecipadamente, apenas retirando seus produtos no balcão. A cantina também dispõe de uma interface administrativa para gerenciar cardápio, pedidos e relatórios.

Nosso objetivo é melhorar a experiência do consumidor e a gestão operacional da cantina, utilizando tecnologias modernas para garantir uma aplicação intuitiva, segura e eficiente.

🛠 Instalação

<b>Android:</b>

Faça o download do arquivo APK do aplicativo no seu celular.
Execute o APK e siga as instruções do dispositivo para instalação.

# Comando para instalar APK via adb (opcional)
adb install ComendariaDaTia.apk

<b>Windows:</b>

Não há instalação necessária, pois o app é mobile.
Entretanto, você pode executar o backend localmente em ambiente Windows.

# Exemplo para rodar backend Node.js
cd src/Backend
npm install
npm start

<b>HTML:</b>

Interface administrativa pode ser acessada via navegador web.
